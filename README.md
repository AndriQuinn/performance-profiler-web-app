# Performance Profiling System 📱 
A very simple benchmarking web appilication for hybrid algorithms

<img src="https://github.com/AndriQuinn/performance-profiler-web-app/blob/main/src/screenshots/home.png?raw=true" width="100%" alt="Banner">


## Features 🚀
- Simple and Clean UI
- Upload dataset or generate data
- Set number of search attempts
- Graph based visualization
- Test Hybrid Algorithms:
  - Interpolation-Binary
  - Interpolation-Fibonacci
  - Interpolation-Exponentai

## Installation / Usage ⬇️

Information:

- React JS
- Version = 1.0.0

1. Live Web Application:
   - Click the [Link](https://performance-profiler-web-app.vercel.app/)
   - Done! 🥳

2. Clone the repository:
      ```bash
        git clone https://github.com/AndriQuinn/performance-profiler-web-app.git
      ```
   - Open in Visual Studio Code
   - Type:
   ```bash
        npm i
      ```
   - And then lastly type:
   ```bash
        npm run dev
      ```
   - Click the address eg. localhost:5173
   - Done! 🥳
